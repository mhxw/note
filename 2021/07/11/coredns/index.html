在Ubuntu上安装CoreDNS

2021-07-11
  • linux
  • dns
  • ubuntu
  • coredns

# 设置

# 1. 从 官网 (opens new window) 载CoreDNS

将二进制文件解压到 /mhxw/dns并使其可执行

sudo chmod +x /mhxw/dns/coredns

docker pull coredns/coredns

mkdir -p /mhxw/coredns
nano /mhxw/coredns/dns/coredns/corefile

# 配置方式

  • 内部hosts
.:53 {
    hosts {
        10.0.0.1 my.host.com
        fallthrough
    }
    forward . 114.114.114.114:53
    log
}
  • 外部hosts
.:53 {
    hosts /etc/coredns/hostsfile {
        fallthrough
    }
    forward . 8.8.8.8:53
    log
}

# cat hostsfile
10.0.0.1 my.host.com

# 启动服务

docker run -it -d --name coredns \
--net=host \
-v /mhxw/dns/coredns:/etc/coredns/ \
coredns/coredns:latest \
-conf /etc/coredns/corefile

# 测试

dig @127.0.0.1 -p 53 my.host.com
