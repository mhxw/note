## Docker 使用教程

docker 停止机器

```shell
docker stop dfd5ea286723
```

docker 删除容器

```shell
docker rm dfd5ea286723
```

docker 删除镜像

```shell
docker rmi f4fd1bb745b2
```

查看docker日志

```shell
sudo docker logs -f bc27c52f48eb9bbfc7146527f92b1de1b9665a7ccbedc59b23f7e3452b049707
```

## Tmux 使用教程

- 安装

```shell
sudo apt-get install tmux
```

- 新建一个指定名称的会话

```shell
tmux new -s prometheus
```
